## Working MuJoCo Model

The current working model is:

```text
mujoco/g0.xml
```

It is generated by:

```bash
TERM=xterm conda run -n g0_isaaclab python scripts/sim2sim/generate_g0_mujoco_from_urdf.py
```

Generation process:

1. MuJoCo loads the source URDF.
2. MuJoCo emits an intermediate MJCF under `mujoco/generated/`.
3. The script rewrites mesh paths relative to `mujoco/g0.xml`.
4. The URDF-derived tree is wrapped in a floating `base_link` body.
5. A ground plane is added.
6. 22 position actuators are added with names matching `G0_JOINT_SDK_NAMES`.

`mujoco/generated/` is an intermediate output directory and should not be used as the committed model source.

## Self-Collision Filtering Update

The URDF-derived working model now applies MuJoCo collision filtering to match Isaac Lab's `enabled_self_collisions=False` setting:
